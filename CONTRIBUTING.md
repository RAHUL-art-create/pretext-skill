# Contributing to Pretext Skills Guide

Please help us make this skill the best and most comprehensive reference for AI agents building with Pretext.

## How to contribute
1. Fork the repo.
2. Add a new **Visual Recipe** or **Core Concept** in `SKILL.md`.
3. Submit a Pull Request.

## Guidelines
- Follow the existing Markdown structure.
- Keep recipes concise and actionable for AI agents.
- Ensure all layout strategies are based on pure arithmetic.
